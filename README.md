# Voice Assistant 

This is a simple Python Voice Assistant that can listen to your voice and respond with spoken output.  
It can:
- Tell the current time
- Open websites
- Search Google
- Speak responses
- And more...

## How to Run

### 1. Install the required libraries
# voice-assistant
pip install -r requirements.txt

insted of PyAudio just install sounddevice scipy

pip install sounddevice soundfile SpeechRecognition pyttsx3 pywhatkit pyjokes


### 2. Run the program
python main.py
